# Google Play Store Data Analysis

A data analysis project exploring the Google Play Store dataset using Python to uncover insights about app categories, ratings, reviews, installs, pricing, and user engagement.

## Project Overview
This project focuses on cleaning, preprocessing, and analyzing Google Play Store data to identify trends and patterns that can support data-driven decisions.

## Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Handling missing values and duplicates
- Data visualization using charts and graphs
- Analysis of app ratings, reviews, installs, categories, and pricing
- Insights into factors affecting app popularity

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Identified the most popular app categories.
- Analyzed the relationship between ratings and installs.
- Compared free and paid applications.
- Explored how reviews and pricing influence app performance.

## Dataset
The project uses the **https://www.kaggle.com/datasets/lava18/google-play-store-apps** dataset from Kaggle.

## Goal
To demonstrate practical data analysis skills by transforming raw Google Play Store data into meaningful insights through data preprocessing, visualization, and exploratory analysis.
